# Jogo da Velha em Python (Console)

Um jogo da velha clássico desenvolvido em Python 3 para rodar diretamente no terminal. O foco do projeto foi aplicar boas práticas de programação, modularização de código e validação robusta de dados de entrada.

### 🚀 Funcionalidades
* **Tratamento de Erros:** Sistema contra jogadas em posições ocupadas ou caracteres inválidos.
* **Interface Limpa:** Atualização dinâmica da tela a cada turno (limpando o histórico do console).
* **Placar Persistente:** Armazenamento e exibição do número de vitórias de cada jogador entre as partidas.
* **Multiplataforma:** Suporte nativo para comandos de limpeza de tela no Windows, Linux e macOS.
